# 契約詳細

## 1. API概要

| 項目           | 内容                             | 備考                     |
| -------------- | -------------------------------- | ------------------------ |
| API名          | 契約詳細                         | -                        |
| エンドポイント | `/api/v1/contracts/{contractId}` | {contractId}は対象契約ID |
| メソッド       | GET                              | リソースの取得           |
| 系統           | 取得                             | -                        |
| 認証要否       | 必要                             | -                        |
| データ形式     | JSON                             | -                        |

## 2. リクエストヘッダ

| 項目名          | 必須 | 値の例             | 備考         |
| --------------- | ---- | ------------------ | ------------ |
| `Authorization` | 必須 | `Bearer eyJ...`    | 認証トークン |

## 3. リクエスト

### パスパラメータ

| 論理名     | 物理名    | 型     | DBテーブル | DBカラム | 必須 | 最小桁数 | 最大桁数 | フォーマット | 最小値 | 最大値 | 備考 |
| ---------- | --------- | ------ | ---------- | -------- | ---- | -------- | -------- | ------------ | ------ | ------ | ---- |
| 契約ID | contractID | string | 契約   | ID       | 必須 | 36       | 36       | UUID         | -      | -      | -    |

## 4. 処理概要

パスパラメーターで指定された契約IDに基づき、該当契約の詳細情報を取得し返却する。認証トークンにより、アクセス権限を確認する。

## 5. レスポンス

- **レスポンスコード** `200 OK`

- **レスポンス**

### ルートオブジェクト(レスポンスボディ)

| 論理名         | 物理名          | 型     | DBテーブル | DBカラム   | 備考                                |
| -------------- | --------------- | ------ | ---------- | ---------- | ----------------------------------- |
| 契約ID     | id              | string | 契約   | ID         | -                                   |
| 契約名     | name            | string | 契約   | 契約名 | -                                   |
| 契約概要   | description     | string | 契約   | 概要       | 詳細な説明文                        |
| 契約区分   | type            | number | 契約   | 区分       | 1: サブスク, 2: 従量課金            |
| 基本料金       | basePrice       | number | 契約   | 基本料金   | 月額料金など                        |
| 開発元会社名   | developerName   | string | 開発元     | 会社名     | -                                   |
| 登録日時       | createdAt       | string | 契約   | 登録日時   | ISO 8601形式 (YYYY-MM-DDTHH:mm:ssZ) |
| 最終更新日時   | updatedAt       | string | 契約   | 更新日時   | ISO 8601形式 (YYYY-MM-DDTHH:mm:ssZ) |
| 関連機能リスト | relatedFeatures | array  | -          | -          | その契約に含まれる機能の配列    |

## 6. エラー定義

| HTTPステータスコード | エラーメッセージ(必須)       | エラーメッセージ詳細(任意) | 発生条件                                          |
| -------------------- | ---------------------------- | -------------------------- | ------------------------------------------------- |
| 400 BAD REQUEST      | パラメーターが不正です       | 契約IDの形式が不正です     | パスパラメーター(契約ID)の形式が不正な場合      |
| 401 UNAUTHORIZED     | 認証情報が無効です           | -                          | 認証が必要なAPIでトークンが無効または未送信の場合 |
| 404 NOT FOUND        | 該当する契約が見つかりません | -                          | 指定された契約IDのデータが存在しない場合          |
