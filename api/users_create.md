# ユーザー作成

## 1. API概要

| 項目             | 内容            | 備考         |
| ---------------- | --------------- | ------------ |
| API名            | ユーザー作成    | -            |
| エンドポイント   | `/api/v1/users` | -            |
| リソース         | `users`         | -            |
| アクション       | `create`        | -            |
| メソッド         | `POST`          | リソース作成 |
| ビジネスドメイン | ユーザー管理    | -            |
| 認証要否         | 不要            | -            |
| データ形式       | JSON            | -            |

## 2. リクエストヘッダ

| 項目名        | 必須 | 値の例           | 備考         |
| ------------ | --- | --------------- | ----------- |
| `Content-Type`  | 必須 | `application/json` | -            |

## 3. リクエスト

### ルートオブジェクト(リクエストボディ)

| 論理名         | 物理名   | 型     | DBテーブル | DBカラム       | 必須 | 最小桁数 | 最大桁数 | フォーマット   | 最小値 | 最大値 | 備考 |
| -------------- | -------- | ------ | ---------- | -------------- | ---- | -------- | -------- | -------------- | ------ | :----- | :--- |
| ユーザー名     | name     | string | ユーザー   | ユーザー名     | 必須 | 1        | 256      | -              | -      | -      | -    |
| メールアドレス | email    | string | ユーザー   | メールアドレス | 必須 | 6        | 256      | メールアドレス | -      | -      | -    |
| パスワード     | password | string | ユーザー   | パスワード     | 必須 | 8        | 16       | -              | -      | -      | -    |

## 4. 処理概要

ユーザー情報を登録する。

## 5. レスポンス

- **レスポンスコード** `201 CREATED`

- **レスポンス**

| 論理名     | 物理名 | 型     | DBテーブル | DBカラム | 備考 |
| ---------- | ------ | ------ | ---------- | -------- | ---- |
| ユーザーID | id     | string | ユーザー   | ID       |      |

## 6. エラー定義

| HTTPステータスコード      | エラーメッセージ(必須)       | エラーメッセージ詳細(任意)       | 発生条件                                   |
| ------------------------- | ---------------------------- | -------------------------------- | ------------------------------------------ |
| 400 BAD REQUEST           | 入力内容に誤りがあります     | メールアドレスを入力してください | バリデーションエラー                       |
| 409 CONFLICT              | このユーザーは登録できません | -                                | メールアドレスが被った場合                 |
| 500 INTERNAL SERVER ERROR | システムエラーが発生しました | -                                | 予期せぬサーバー内部エラー(DB接続失敗など) |
